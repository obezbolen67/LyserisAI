# Lyseris AI Frontend

Web and mobile-ready chat UI for Lyseris AI. Fast React + Vite app with voice chat, tool blocks, and Stripe checkout.

## Setup

Requirements: Node 18+

1) Install
```bash
npm install
```

2) Create `.env`
```bash
VITE_API_URL=http://localhost:3001
VITE_STRIPE_PUBLISHABLE_KEY=pk_test_...
GOOGLE_MAPS_API_KEY=AIza...
```

3) Run
```bash
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Notes

- For Android packaging, use Capacitor scripts in `package.json`.
- In production, build and serve the static files from a web server or CDN.
